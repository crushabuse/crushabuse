crushabuse

c'est un script qui t'aide a trouver des amants

il est fait en 2 parties

  • server

  • client (uses libnotify, node, etc)

prerequisites

# sox est optional, sauf si vous voulez le son de notification :p
# vous pouvez exclure libnotify & terminal-notifier si le node.js est installé
sudo dnf install libnotify sox # debain/ubuntu 
sudo apk add libnotify sox # alpine based, ish, termux, etc
sudo dnf install libnotify sox # fedora and other red hat distros
brew install terminal-notifier sox # macos
# check https://pkgs.org/search/?q=libnotify if yours isn't listed here

install 

wget -O /bin/crushabuse https://crushabuse.xyz/client.sh
chmod +x /bin/crushabuse

usage

Ouvrez crushabuse, assurez-vous d'avoir NAME à votre nom et vous pouvez activer SOUND, mais consultez le code pour plus de details.

Ceci est cense s'executer en tant que démon (processus en arrière-plan), mais il ne vérifie pas automatiquement, uniquement lors de l'invocation.

C'est un petit hack pour le faire fonctionner toutes les 5min en arriere-plan (necessite un nœud).

nohup node -e "function main(){ require('node:child_process').execSync('bash ./client.sh')};main();setInterval(main, 1000* 60 * 5)" > /dev/null 2>&1 &

Vous pouvez egalement utiliser crontab(5), mais nous ne savons pas si les notifications fonctionneront correctement mdr
